Broker Dynamic API Postman collection

## Ерөнхий урсгал

1. `POST /api/broker/get_token`
- Broker username, password-оор token авна.
2. `POST /api/broker/get-driver`
- Регистрийн дугаараар жолооч / иргэний мэдээллийг татна.
- XYP ашигладаг.
- Зөвхөн XYP ашиглах эрхтэй хэрэглэгч ашиглана.
3. `POST /api/broker/get-vehicle`
- Улсын дугаараар тээврийн хэрэгслийн мэдээллийг татна.
- XYP ашигладаг.
- Зөвхөн XYP ашиглах эрхтэй хэрэглэгч ашиглана.
4. `GET /api/broker/products`
- Ашиглах боломжтой бүтээгдэхүүний жагсаалт авна.
5. `GET /api/broker/products/{product_key}/category`
- Сонгосон `product_key`-ийн category авна.
- Буцах category: `mashin`, `oron_suuts`, `ed_hurungu`.
6. `GET /api/broker/{product_key}/get_sets`
- Сонгосон бүтээгдэхүүний багцуудыг авна.
7. `POST /api/broker/{product_key}/create`
- Сонгосон бүтээгдэхүүний гэрээг үүсгэнэ.
8. `POST /api/broker/{product_key}/upload_photos`
- Гэрээ үүссэний дараа тээврийн хэрэгслийн зургуудыг upload хийнэ.
- Зөвхөн `thd`, `ht`, `thd_loan` дээр ажиллана.
9. `POST /api/broker/{product_key}/check_pay`
- Төлбөрийн төлөвийг шалгана.
- Төлөгдсөн бол `contract_url` буцаана.
10. `POST /api/broker/{product_key}/contract`
- Нэг гэрээний дэлгэрэнгүй мэдээллийг авна.
11. `POST /api/broker/{product_key}/contracts`
- Огнооны интервалаар гэрээнүүдийн жагсаалтыг авна.

## Нэмэлт тайлбар

- Тээврийн хэрэгслийн даатгал нь хуучин `/api/broker/thd/*` flow-ийг яг тэр чигээр нь ашиглана.
- Хүнд даацын машины даатгал, Гэнэтийн ослын даатгал, Зээлтэй автомашины даатгал, Албан журмын даатгал, Суудлын гэнэтийн ослын даатгал нь broker dynamic controller-оор дамжина.
- Албан журмын даатгалын `get_sets` нь хоосон жагсаалт буцаана.
- Тээврийн хэрэгслийн даатгалын `get_sets` нь хуучин THD sets response-ээ яг тэр чигээр нь ашиглана.
- Гэнэтийн ослын даатгал, Суудлын гэнэтийн ослын даатгалын `get_sets` нь common broker structure дээр `insurance_type`, `protection_amount`, `price` талбаруудыг нэмээд буцаана.
- Баталгаажсан token-ийг `Authorization: <token>` хэлбэрээр protected endpoint бүрд дамжуулна.
- `set` утга нь тухайн product-ийн `get_sets` endpoint-ээс авсан key/id байна.
